Design and Implementation of a Secure Document Exchange System

dipeshkc.pythonanywhere.com

HOW TO USE THE SYSTEM
1 Register —Create a free account with your email and password —hashed and stored securely
2 Upload —Click Send Document, enter recipient email, choose any file —it encrypts automatically
3 Receive —Recipient logs in and sees the document on their dashboard with pending status
4 Approve —Recipient views, downloads the decrypted file, and clicks Approve to confirm receipt
5 Track —Sender sees full audit log —who opened it, when, and how many time
